<h2>⚙️ Run Locally</h2>

<p>Clone the repository</p>

<pre>
git clone https://github.com/yourusername/portfolio.git
</pre>

<p>Go to frontend directory</p>

<pre>
cd src/frontend
</pre>

<p>Install dependencies</p>

<pre>
pnpm install
</pre>

<p>Run development server</p>

<pre>
pnpm vite
</pre>

<p>Open browser</p>

<pre>
http://localhost:5173
</pre>
